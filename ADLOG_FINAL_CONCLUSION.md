## 1. 핵심 발견

### N1, N2, N3의 정체

| 지수 | 정체 | 특성 |
|------|------|------|
| **N1** | 키워드 지수 | 키워드별 고정값 (예: 홍대맛집=0.366894) |
| **N2** | 품질 지수 | **업체 고유값** - 어떤 키워드로 검색해도 동일 |
| **N3** | 종합 지수 | N1, N2로 계산됨. **순위 = N3 내림차순** |

### 충격적 사실

```
"카페 공명 홍대점"의 N2 값:
  - "홍대맛집" 검색: N2 = 0.543409
  - "홍대카페" 검색: N2 = 0.543409 (동일!)

→ N2는 업체 고유의 "품질 점수"
→ V, B, S와 무관하게 결정됨
```

## 2. V, B, S로 N2 예측 불가

| 모델 | R² | 해석 |
|------|-----|------|
| log(V) + log(B) + log(S) | 2.7% | 거의 설명 못함 |
| + 상호작용항 | 19.3% | 여전히 낮음 |
| **결론** | **< 20%** | **예측 불가능** |

### 충격적 사례

```
홍대맛집 순위:
  1위 유메오뎅:     V=1,053   B=371    S=2,000   → N2=0.5478
  7위 럭럭마라도원:  V=600     B=35     S=100     → N2=0.5472

→ 7위가 V, B, S 모두 작지만 N2는 1위와 거의 같음!
→ V, B, S는 순위와 무관
```

## 3. N2의 정체 추정

N2를 결정하는 숨겨진 요소 (90% 이상):

| 요소 | 설명 | 접근 가능 여부 |
|------|------|---------------|
| CTR (클릭률) | 검색 → 클릭 비율 | 네이버만 접근 가능 |
| 체류시간 | 상세페이지 머무른 시간 | 네이버만 접근 가능 |
| 전환율 | 예약/전화/길찾기 클릭률 | 네이버만 접근 가능 |
| 재방문율 | 반복 방문 비율 | 네이버만 접근 가능 |
| 최근성 | 최근 리뷰/활동 빈도 | 크롤링 가능 |

## 4. ADLOG의 데이터 소스 추정

### 가설 A: 네이버 비공개 API 접근 (가능성 높음)
- ADLOG가 네이버 파트너로서 내부 API 접근권 보유
- N2 값을 직접 받아옴

### 가설 B: 크롤링 + 자체 알고리즘
- 네이버 플레이스 페이지 크롤링
- 숨겨진 데이터 추출 또는 자체 추정

### 가설 C: 네이버 내부 협력
- 네이버 직원 또는 협력사와 협력
- 내부 데이터 공유

**결론: 외부에서 N2 계산 로직을 알아내는 것은 불가능**

## 5. 순위 결정 공식

```python
# 확인된 공식
순위 = N3 내림차순  # Spearman 상관 = 1.000 (완벽)

# N3 계산
N3 = f(N1, N2)  # 정확한 공식은 ADLOG 내부에만 존재
# 단순 근사: N3 ≈ 0.05 * N2 + 0.34
```

## 6. 현실적 해결책

### 방법 1: ADLOG API 직접 활용 (권장)

```python
import requests

def get_adlog_data(keyword):
    response = requests.post(
        "http://adlog.ai.kr/placeAnalysis.php",
        json={"query": keyword},
        headers={"Content-Type": "application/json"}
    )
    return response.json()

# 사용
data = get_adlog_data("홍대맛집")
for item in data["items"]:
    print(f"{item['place_rank']}위: {item['place_name']}")
    print(f"  N1={item['place_index1']}, N2={item['place_index2']}, N3={item['place_index3']}")
```

### 방법 2: N2 변화 모니터링

```
1. 매일 ADLOG API 호출
2. N2 값 변화 추적
3. N2 상승 → 순위 상승 예측
4. 경쟁사 N2 분석으로 전략 수립
```

### 방법 3: 행동 지표 개선 (N2 상승 유도)

| 행동 | 예상 효과 |
|------|----------|
| CTR 개선 (제목/이미지 최적화) | N2 상승 |
| 체류시간 증가 (상세 정보 제공) | N2 상승 |
| 전환율 향상 (예약/전화 유도) | N2 상승 |
| 재방문 유도 (단골 마케팅) | N2 상승 |
